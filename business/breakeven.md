# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute**: $0.15/user/month
- **Storage**: $0.05/user/month
- **Bandwidth**: $0.10/user/month
- **Total Cost per Active User**: 
  - **Total** = Compute + Storage + Bandwidth
  - **Total** = $0.15 + $0.05 + $0.10 = **$0.30/user/month**

### Pricing Tiers
| Tier Name       | Price ($/mo) | Features                                                                 |
|------------------|--------------|--------------------------------------------------------------------------|
| **Basic**        | $10          | Real-time transaction status updates, Basic troubleshooting alerts       |
| **Pro**          | $25          | All Basic features + Advanced analytics, Custom alerts, Email support   |
| **Enterprise**   | $50          | All Pro features + Dedicated account manager, API access, SLA guarantees |

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$50 - $100** per user

### Lifetime Value (LTV) Estimate
- Average Monthly Revenue per User (ARPU) for each tier:
  - Basic: $10
  - Pro: $25
  - Enterprise: $50
- Average user retention: 24 months
- LTV Calculation:
  - Basic: $10 * 24 = **$240**
  - Pro: $25 * 24 = **$600**
  - Enterprise: $50 * 24 = **$1200**

### Break-even Users Count
- Fixed Costs (assumed): $5,000/month
- Contribution Margin per User:
  - Basic: $10 - $0.30 = $9.70
  - Pro: $25 - $0.30 = $24.70
  - Enterprise: $50 - $0.30 = $49.70
- Break-even Users Count Calculation:
  - Basic: $5,000 / $9.70 ≈ **517 users**
  - Pro: $5,000 / $24.70 ≈ **202 users**
  - Enterprise: $5,000 / $49.70 ≈ **101 users**

### Path to $10K MRR
- Target Monthly Recurring Revenue (MRR): $10,000
- Pricing Tier Strategy:
  - **Basic Tier**: 
    - Users needed: $10,000 / $10 = **1,000 users**
  - **Pro Tier**: 
    - Users needed: $10,000 / $25 = **400 users**
  - **Enterprise Tier**: 
    - Users needed: $10,000 / $50 = **200 users**

### Summary
To achieve $10K MRR, we can target different tiers based on user acquisition strategy:
- **1,000 Basic users**
- **400 Pro users**
- **200 Enterprise users** 

This analysis provides a clear path to profitability and helps in strategizing user acquisition efforts.
