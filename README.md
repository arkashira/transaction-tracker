# Transaction Tracker

A simple transaction tracker that allows you to add transactions and filter them by date range, merchant ID, and status.

## Usage

1. Create a `TransactionTracker` object.
2. Add transactions using the `add_transaction` method.
3. Get transactions using the `get_transactions` method.
4. Get failed transactions using the `get_failed_transactions` method.

## Testing

Run the tests using `pytest`.
